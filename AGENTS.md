# Design System Rules
- **Framework:** Next.js + Tailwind + Shadcn
- **Font Stack:** Fontshare (Satoshi for UI, General Sans for headings)
- **Visual Style:** High-end, minimal, dark-mode first.
- **Reference Gallery:** Use patterns from [Godly.website] & [Minimal.gallery].

# Component Resources (Only fetch when needed)
- Registry: https://21st.dev (For complex landing sections)
- Animations: Rive (for state machines), Reactbits (for hover effects)
- 3D: Spline (for hero backgrounds)

# Instructions
1. NEVER use standard CSS; use Tailwind.
2. If building a new section, check `npx shadcn search` first.
3. For motion, prioritize Framer Motion over basic transitions.
